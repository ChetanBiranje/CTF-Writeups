# Contributing to CTF Writeups

Thank you for considering contributing! 🎉

## How to Contribute

### Adding Your Writeup

1. Fork this repository
2. Create a new writeup using TEMPLATE.md
3. Place it in the appropriate category folder
4. Follow the writeup format
5. Submit a pull request

### Guidelines

**Format:**
- Use the provided template
- Include all sections
- Add code blocks with syntax highlighting
- Include screenshots if helpful

**Quality:**
- Clear explanations
- Step-by-step solutions
- Explain WHY, not just HOW
- Include learning points
- Add references

**Organization:**
- Place in correct category
- Use descriptive filename
- Tag appropriately

### What to Include

✅ Challenge description  
✅ Tools used  
✅ Detailed solution  
✅ Key learnings  
✅ References  
✅ Flag (spoiler-tagged if needed)  

### What NOT to Include

❌ Writeups during active CTF  
❌ Plagiarized content  
❌ Incomplete solutions  
❌ Overly complex explanations  

## Questions?

Open an issue or contact me!

Thank you! 🙏
